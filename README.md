# Prueba full stack

Tecnologias usadas para realizar esta prueba

Frontend:
- React
- ant-design
- node sass
- react-router-dom

BackEnd:
- NodeJs
- bcrypt
- body-parser
- connect-multiparty
- express
- jwt-decode
- moment
- mongoose
- nodemon

Base de Datos :
- MongoDb

Gestor de paquetes:
- Yarn

-Servicio utilizados para subir la aplicacion a la nube:
- Mongo Atlas
- Heroku
- Netlify
